Simprolog is an implementation of a simple version of Prolog in OCaml. This project includes a grammar, lexer, parser, semantics, evaluator (including back-tracking).

File list:
* parser.mly -- Parser
* lexer.mll  -- Lexer
* main.ml    -- Main functionality including unification and backtracking.
* simp.ml    -- Interactive interface

Installation
* Instal OCaml 4.03.0
* run script "mk" by executing ". mk" in terminal.
* run executable simp by "./simp" interminal.

For examples and implementation details, readers are encouraged to read report/report.pdf for more information.
